REPOSITORIO DE LA ASIGNATURA DE BASE DE DATOS II

Explora el fascinante mundo de las bases de datos avanzadas, arquitecturas distribuidas y administración de sistemas de información

Contenido

El curso Base de Datos II consiste en el estudio avanzado de sistemas de gestión de bases de datos, enfocado en arquitecturas distribuidas, administración de servidores, seguridad de datos y optimización del rendimiento. A lo largo de 16 semanas, los estudiantes desarrollarán competencias para diseñar, implementar y administrar sistemas de bases de datos empresariales escalables y seguros.

Portafolio

Este portafolio recopila los proyectos y actividades desarrollados durante el curso, incluyendo diseños de arquitecturas de bases de datos, implementación de instancias de servidores, estrategias de seguridad y planes de backup y recuperación. Cada semana representa un avance significativo en el dominio de las tecnologías de bases de datos.
